# Maquetación Web: HTML 5 y CSS
Curso de Maquetación Web: HTML 5 y CSS

## Clase 6 - Integrando Conocimientos

## Objetivos
En esta clase veras:
* Trabajarás con la regla \@media
* Implementarás Bootstrap básico en tu proyecto Integrador
* Jerarquías y prioridades

Al final de este modulo seras capaz de:
* Integrarás conocimientos
* Terminarás este curso maquetando
* Trabajar con todos los elementos visto respetando jerarquías y priorizando elementos

* Apuntes

* Manuales
* Presentacion
* Videos
